## Lab 1: SOC Fundamentals - Investigating a Port Scan Alert (False Positive)
This lab simulates a level 1 SOC analyst investigating a high severity port scan alert using SIEM tools. Through context review and log analysis i determined it authorized activity and closed it as a false positive.

**Date:** January 7, 2026

**Platform:** TryHackMe (SOC fundementals SIEM alert lab)

## Objective
Review SIEM and verify Port scanss marked as high severity in a simulated SOC environment.

## Steps Taken
1. High severity Port Scan alert Acknowledged and reviewed 
2. Analyzed network traffic logs in the SIEM
3. Identified source IP, destination, host names, timestamps, and activity patterns
4. Reviewed provided context regarding internal vulnerability assessment activities
5. Classified the alert as a false positive due to authorized internal scanning


## Skills demonstrated
- Showed understanding of the 5 W's
- Security Monitoring and alert analysis
- Incident Triage and reporting
- False positive identification and alert disposition

## Results
The alert was proven to be a false positive incident, the vulnerability assesment team was running a port scan activity.
