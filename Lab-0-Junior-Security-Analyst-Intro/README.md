## Lab 0: Blocking a malicious IP

**Date:** January 7, 2026

**Platform:** TryHackMe (Junior Security Analyst intro Lab)

## Objective
Review a SIEM and isolate a malicious IP tagged as a critical alert in a simulated SOC environment.

## Steps taken
1. Monitored alerts on the SIEM system.
2. Malicious IP attempting unauthorized logins.
3. Escalated the event to senior analyst.
4. Blocked the malicious IP.
5. Verified that the network was safe.
6. Documented response and findings.

## Skills demonstrated
- Security Monitoring and alert analysis
- Firewall configuration and rule implementation 
- Incident triage and reporting

## Results
Malicious IP successfully blocked. confirmed no further unauthorized traffic.
